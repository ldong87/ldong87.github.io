  1. ๏ Zestimate                                                                                                                                                                Aug. - Oct. 2017

    This is a Kaggle contest that tries to help improve Zillow’s current home value estimation algorithm. Our team is currently ranked top 1% out of ~3800 teams. I created a meta-learning algorithm that make use of most of popular machine learning algorithms. There are two levels of learners. The first level includes gradient boosting, SVM, kNN, stochastic elastic net, fast linear online learning, genetic programming and neural networks. They take input from preprocessed structured training data (160k+ entries and each has tens of features). The second level learners include a couple of gradient boosting algorithms. They take input from the training data as well as the output from the first level learners as meta-features. This meta-feature based algorithm can boost performance significantly. This framework is coded in Python (+1000 lines) and differs from existing stacking framework by considering the temporal pattern in the data. This framework uses machine learning libraries such as XGB, LightGB, CatBoost, Scikit Learn, Vowpal Wabbit, etc. The code is to be released.

  1. ๏ Hospital utilization prediction (project repo)                                                                                                                     Apr. 2017

     This is a hackathon hosted by Lally School of Management and IDEA at RPI. Our team won the third prize out of ~28 teams.   1. ๏ Twitter sentiment analysis (project repo)                                                                                                                Feb. - May 2017

     This is a course project for Deep Learning. 500,000 tweets are preprocessed (mainly masking and word embedding with word2vec) and serve as input to recurrent neural networks. In order to determine whether a tweet is positive or negative, 500k tweets are preprocessed (mainly word embedding with word2vec) and fed to recurrent neural networks (RNN) since RNN is natural for 1D time series data.
